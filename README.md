Creates wallpapers based on the albums you like using the spotify api to get your liked albums and the itunes api to get the highest image quality possible.


For this program to work you need to provide it a spotify client id and client secret, you can find this by going on the spotify developer dashboard and creating an application, set the callback url to "http://localhost:8888/callback"  then just follow the steps provided when running the program
